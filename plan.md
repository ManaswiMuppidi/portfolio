Proposed Changes
1. Project Initialization & Structure
We will establish a simple file structure for the vanilla web app:

index.html: The main structural file containing all sections.
css/style.css: Core design system, color variables (neon blue, purple, cyan), glassmorphism classes, and typography.
css/animations.css: Keyframe definitions for continuous floating, levitating, and scroll-triggered animations.
js/main.js: Intersection Observers for scroll animations and interactive logic (e.g., hover effects).
assets/: Directory for generated images, graphics, and icons.
2. Core Themes and Styling
Color Palette: Dark space background (almost black with subtle gradients) with glowing accents in Blue (#00f3ff), Purple (#bc13fe), and Cyan.
Typography: Crisp, tech-focused fonts (e.g., 'Orbitron' for headers, 'Inter' for body).
Glassmorphism: Panels and cards will have background: rgba(255, 255, 255, 0.05), backdrop-filter: blur(10px), and subtle white/neon borders.
Animations: Elements will use a CSS animation: float 6s ease-in-out infinite to create the zero-gravity effect.
3. Section Implementations
Hero Section: Large dramatic headers, animated space/data particles background, floating Call to Action (CTA) buttons.
About Section: Glass cards with floating effects containing personal and technical background.
Skills Section: Staggered, floating elements for programming languages, tools, and core skills with glowing progress indicators.
Projects Section: 4 levitating glass cards with neon hover effects detailing the user's data-driven projects.
Dashboard Section: A visual highlight featuring floating "holographic" data widgets (bar charts, line graphs, KPIs). We will simulate the charts using CSS and/or generated imagery for a truly sci-fi feel.
Resume & Certifications: Levitating document panels and badge icons.
Contact & Footer: Glowing input fields, animated send button, and a clean minimalist footer.
